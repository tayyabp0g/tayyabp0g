## Tech Stack

- **Next.js**: A powerful React framework for server-rendered applications.
- **Node.js**: A runtime environment for executing JavaScript on the server-side.
- **AI/ML Libraries**: Comprehensive documentation on the following libraries:
  - TensorFlow: Open-source library for deep learning.
  - PyTorch: An open-source machine learning framework.
  - scikit-learn: Simple and efficient tools for data mining and data analysis.
  - Keras: High-level neural networks API, written in Python.
  - OpenAI: Provides APIs for a wide array of AI tasks.
